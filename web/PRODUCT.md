# Product

## Users

Vietnamese health-conscious individuals who want to:

- Track their weight over time
- Get personalized calorie recommendations based on weight trends
- Plan meals using authentic Vietnamese recipes

Context: Daily use, likely checking weight updates and meal plans. Mobile-friendly use case.

## Product Purpose

A meal planning app that adapts calorie suggestions based on user's weight trends, enriched with a Vietnamese dish database. Users update weight periodically, and the app adjusts recommendations accordingly.

Core value: "Adaptive meal planning + Vietnamese cuisine identity"

## Brand Personality

- **Warm & practical** - like a helpful Vietnamese home cook
- Approachable, not clinical
- Trustworthy nutrition data
- Celebrates Vietnamese culinary heritage

## Anti-references

- Generic AI-generated aesthetics
- SaaS "cream" color schemes
- Corporate fitness app vibes (no navy-and-gold)
- The "eyebrow + numbered sections" landing page cliche
- Gradient text decorations

## Design Principles

1. **Utility first** - Every UI element serves a functional purpose
2. **Vietnamese identity through content, not decoration** - The cuisine is the feature, not just a visual theme
3. **Adaptive intelligence** - Weight tracking should feel smart, not just data entry
4. **Warm clarity** - Soft, readable, no harsh contrasts or shouting
5. **Progressive simplicity** - Features unfold as user engages, not overwhelming at first

## Accessibility & Inclusion

- Standard WCAG 2.1 AA compliance
- Support system font scaling
- Reduced motion respect for users who prefer less animation
- Clear contrast ratios for all text (≥4.5:1 for body)
